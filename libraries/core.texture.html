set_texture (entity, texture[, visual])
set texture on entity

Parameters:

  • entity LuaEntity LuaEntity on which texture should be set
  • texture string name of the texture file to be set on entity
  • visual string set visual. If not provided, entity original visual will be used (optional)

Returns:

    bool true
exists (name[, directory])
check if texture with given name exists

Parameters:

  • name string texture file name
  • directory string name of subdirectory inside textures directory. By default path (optional)

Returns:

  1. bool true texture file exists
  2. bool false texture not exists
download (url, secure, name[, directory])
download texture from url save and load

Parameters:

  • url string url of the texture file
  • secure bool is url uses https
  • name string name of texture file to be saved
  • directory string name of subdirectory inside path (optional)

Returns:

    bool true (successfully downloaded and saved)

Or

  1. nil
  2. string error message
download_from_9p (conn, source_path, name[, directory])
download texture over 9p

Parameters:

  • conn conn 9p connection
  • source_path string path to the texture
  • name string name of texture file to be saved
  • directory string name of subdirectory inside path (optional)

Returns:

    bool true (successfully downloaded and saved)

Or

  1. nil
  2. string error message
